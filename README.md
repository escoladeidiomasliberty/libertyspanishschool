# Liberty Spanish School — GitHub Pages Website

This folder contains a premium, responsive, bilingual (EN/ES) landing page:
- **index.html**
- **styles.css**
- **script.js**
- **assets/** (logo + background)

## 1) Replace your links (Stripe, Telegram, Calendly/WhatsApp)
Open **script.js** and replace these placeholders:

- `https://YOUR_STRIPE_LINK_HERE`
- `https://YOUR_TELEGRAM_LINK_HERE`
- `https://YOUR_CALENDLY_OR_WHATSAPP_LINK_HERE`

Optionally update:
- Instagram / TikTok / YouTube links / 

## 2) Replace your logo
Replace `assets/logo.svg` with your real logo:
- Keep the filename the same: `logo.svg`
- Or update the `<img>` src in index.html accordingly.

## 3) Publish on GitHub Pages (no coding skills needed)

### A) Create a repository
1. Go to GitHub → **New repository**
2. Name it: `libertyspanishschool` (or any name)
3. Create repository

### B) Upload files
Upload:
- index.html
- styles.css
- script.js
- assets/ folder

### C) Enable GitHub Pages
1. Repository → **Settings**
2. **Pages**
3. Source: **Deploy from a branch**
4. Branch: **main** / folder: **root**
5. Save

GitHub will give you a public URL.

## 4) Analytics (optional)
You can add Google Analytics (GA4) later by pasting the snippet into `<head>` in index.html.

---

If you want to customize colors, typography, or sections, edit `styles.css` and the text in `script.js`.
